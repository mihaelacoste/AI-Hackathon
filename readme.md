## Create an Expense tracker – CRUD plus charts
• Goal: track expenses with create, edit, delete, filter, and simple charts.
• Starter steps
	• In memory data: amount, category, date, description, tags.
	• Views: add expense, list, filter, export CSV.


How to run:
export GEMINI_API_KEY='API_KEY_here'
streamlit run app.py
